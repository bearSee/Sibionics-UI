# sibionics-ui

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Lints and fixes files
```
npm run lint
```

### Compiles and minifies for production
```
npm run build
```

```
npm run build:utils
```

```
npm run build:theme
```

### Build Common and UMD to pulish
```
npm run dist
```
